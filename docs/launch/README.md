# Launch Readiness

This folder is for public-launch preparation. Keep claims here conservative until they are backed by a live demo, current pricing checks, or source references.

## Approved Positioning

Novyx Vault is an open-source markdown workspace with persistent AI memory, rollback, BYOK providers, desktop-local notes, and optional cloud mode.

## Public Claims Policy

- Avoid "only", "first", "best", "never", and market-size claims unless they are sourced and dated.
- Describe desktop support as a Tauri desktop app/source build unless release binaries have been tested for the target platforms.
- Describe offline support as "core note editing works offline in desktop/local mode"; AI features may require network access unless configured with a local provider.
- Describe provider support as "21 hosted and local provider presets" instead of promising every listed provider works in every environment.
- Do not publish staged or mocked memory demos as proof of product behavior.

## Demo Requirements

A public demo should use real app behavior, not mocked Playwright responses.

- Fresh local vault or disposable cloud account.
- Real notes created through the UI or API.
- Real memory writes and recalls through the configured Novyx memory API.
- Real Ghost Connections, audit, rollback, and weekly review screens if shown.
- No live secrets, personal notes, or production customer data.
- Console reviewed for unexpected errors before recording.

## Assets To Prepare

- Hero screenshot of the editor with preview/backlinks.
- Screenshot of memory timeline/audit with non-sensitive demo data.
- Screenshot of Ghost Connections with real generated connections.
- 60-90 second screen recording proving memory persistence across a reload/new session.
- One-page investor/product brief with dated metrics only.
